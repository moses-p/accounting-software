# Accounting Software

A modern accounting software application built with Next.js, React, and Tailwind CSS.

## Features
- Customer management
- Invoicing
- Expense tracking
- Payroll
- Reports

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- pnpm (or npm/yarn)

### Installation
```bash
pnpm install
```

### Development
```bash
pnpm dev
```

### Production Build
```bash
pnpm build
pnpm start
```

## Testing
```bash
pnpm test
```

## Deployment
- You can deploy this app to Vercel, Netlify, or any platform that supports Next.js.
- For Docker or custom deployments, see the deployment section below.
